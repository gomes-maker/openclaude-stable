Pense em voz alta.

Claude com janela nativa pra macOS. Multi-LLM, 100% local, Liquid Glass. Sem instalar Node, sem terminal — abriu, configurou, conversa.

↓ Baixar pra macOS
108 MB · macOS 26+ · Apple Silicon · v1.0
Instalação
01
Abra o .dmg e arraste o OpenClaudeApp pra pasta Applications.
02
Primeira vez: em Applications, clique com botão direito no app → Open → confirme Open no diálogo. Depois abre normal pelo Launchpad.
03
Configure seu provider de IA no setup (Claude Max, GPT, Gemini, Qwen, DeepSeek). Pronto pra usar.
Recursos

Multi-LLM

Claude · GPT-5 · Gemini · Qwen3 · DeepSeek · Llama. Troca de modelo com ⌘M. Fallback automático via OpenRouter.

100% Local

Conversas só no seu Mac. API keys no Keychain do macOS. Zero telemetria, zero analytics, zero servidor nosso.

Liquid Glass nativo

Identidade visual macOS 26 Tahoe. Bubbles glass, palette com âmbar, transições spring. Reduce Motion respeitado.

Self-contained

Node + CLI bundled dentro do app. Sem brew install, sem npm, sem terminal pra preparar nada.
